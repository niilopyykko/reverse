USAGE:
#INSTALL
gcc -Wall -Wextra -std=c11 -g reverse.c -o reverse

#RUNNING

1 ARGUMENT
./reverse 
After writing a word, hit enter to enter the next one.
ctlr+d ends input loop

2 ARGUMENTS
./reverse input.txt

3 ARGUMENTS
./reverse input.txt output.txt

##REQUIREMENTS

3 ARGUMENTS NEEDS FILES
input.txt output.txt
input.txt should have the text with words seperated by newline character
output.txt should be empty
